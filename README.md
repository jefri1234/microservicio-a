## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository with RabbitMQ integration for asynchronous messaging.

## Project setup

```bash
$ npm install
```

## DOCKER

```bash
$ docker run -d --hostname my-rabbit --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3-management
```

## EXECUTE SERVICE

```bash
$ npm run start:dev
```
